Client project for Ericsson (This Project is live). Working in an AGILE framework with 2-week sprints concluded by a demo at the end of each cycle.

The Problem

The Solution

A user friendly software which gives role-level access to case data and related entities. We developed a complex data structure as well as interfaces to transfer all the unstructured data into exploitable data structures. We digitised their business flow and worked hand-in-hand with several legal departments across multiple countries to consolidate the process. From the database in SQL Server all the way to UI supported by SAIL, our team built the application as well as infrastructure behind it. SSO log-in was also provided.
